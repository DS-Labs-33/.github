## Overview

DS Labs is a specialized digital product studio founded by [Sahaj Italiya](https://github.com/sizwinz) and [Divy Viradiya](https://github.com/divyviradiya2). We focus on building scalable, type-safe, and highly efficient software solutions. Our work bridges the gap between complex engineering requirements and refined user experiences, transforming technical concepts into production-ready deployments.

## Featured Projects

### [Campus Assistant](https://github.com/DS-Labs-33/Campus-Assistant)
A language-agnostic AI chatbot designed for educational institutions to automate routine administrative queries. It handles complex routing workflows across 7+ regional Indian languages.
* **Stack:** Python, FastAPI, Next.js, Google Gemini AI, LangChain, ChromaDB
* **Key Architecture:** Production-ready Retrieval-Augmented Generation (RAG) pipeline, zero-cost translation infrastructure via deep-translator, structured conversation logging, and human-in-the-loop escalation routing.
* **Recognition:** TechSprint Hackathon Entry

### [Blue Can Coffee](https://github.com/DS-Labs-33/Blue-Can-Coffee)
A premium, mobile-first digital identity and commerce infrastructure developed for a specialty coffee brand.
* **Stack:** React, TypeScript, Tailwind CSS, Framer Motion, Shadcn/UI
* **Key Architecture:** Highly optimized responsive layout, integrated JSON-LD schema mapping for hyper-local SEO discovery, and low-latency interaction handling.
* **Status:** [Live Deployment](https://bluecancoffee.vercel.app/)

### [Enso Coffee](https://github.com/DS-Labs-33/enso-coffee)
A sleek, atmospheric single-page digital experience for an artisanal specialty coffee house.
* **Stack:** React, TypeScript, Tailwind CSS, AOS (Animate on Scroll)
* **Key Architecture:** Cinematic layout structures, custom localized location intelligence components, and dynamic menu filtering configurations.
* **Status:** [Live Deployment](https://ensocoffee.vercel.app/)

### [DS Labs Portfolio](https://github.com/DS-Labs-33/ds-labs-portfolio)
The official architectural showcase and operational baseline for the studio.
* **Stack:** Next.js (App Router), React, TypeScript, Tailwind CSS, Web3Forms
* **Key Architecture:** Custom dark/light design system with state persistence, secure backend-less form handshakes, and strict component modularity.
* **Status:** [Live Deployment](https://ds-labs-portfolio.vercel.app/)
